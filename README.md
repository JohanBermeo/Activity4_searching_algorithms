# Algoritmos de Ordenamiento

Este repositorio contiene la implementación de cuatro de los algoritmos de ordenamiento más fundamentales en las ciencias de la computación. Para mantener un entorno de desarrollo limpio y especializado, **la métrica de cada algoritmo se encuentra alojado en su propia rama (branch) dedicada**.

---

## Estructura del Repositorio

La rama principal (`main` o `master`) contiene el código fuente y la implementación. Para revisar las pruebas de un algoritmo específico, debes cambiarte a la rama correspondiente:

| Rama (Branch) | Algoritmo |
| :--- | :--- |
| `algorithm/quicksort` | **Quicksort** |
| `algorithm/shell` | **Shellsort** |
| `algorithm/burbuja` | **Bubble Sort** |
| `algorithm/radix` | **Radix Sort** |
