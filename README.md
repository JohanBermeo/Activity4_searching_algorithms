# Bubble Sort 

Este directorio contiene la implementación, pruebas y análisis de rendimiento del algoritmo **Bubble Sort**.

---

## Descripción del Algoritmo

**Bubble Sort** (o método de la burbuja) es un algoritmo de ordenamiento simple basado en la comparación directa. Funciona revisando repetidamente la lista, comparando elementos adyacentes e intercambiándolos de posición si están en el orden incorrecto. Este proceso de "burbujeo" hace que los elementos más grandes se desplacen gradualmente hacia el final del arreglo en cada pasada. Debido a su complejidad promedio de $O(n^2)$, su uso es principalmente didáctico y no se recomienda para grandes volúmenes de datos.
